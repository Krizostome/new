# Application Web Parc Automobile PAC

Application Web du projet Parc Automobile PAC

## Installation

**Prérequis :**
* NodeJS [Installer NodeJs](https://nodejs.org/en/download/package-manager)
* Un éditeur de code (Visual Studio Code par exemple) [Installer Visual Studio code](https://code.visualstudio.com/download)

**Cloner le projet web :**
```bash
git clone https://github.com/bescportcotonou/parcautopacweb.git
```
<br>

**Installer les dépendances :**
```bash
npm install
```
<br>

**Lancement du projet :** <br>
```bash
ng serve --open
```
